# OptoRslv1
Making a disc pattern for optical resolver

How to use
1. Put all files to a folder.
2. Compile "test.cpp" to make "test.exe". (Edit it before compile if you want to change something)
3. Edit "test.bat" as you like. (Set specifications of the disc here)
4. Execute "test.bat" to make the gerber data.
5. Put gerber data to a viewer.
6. Print it or send it to a PCB maker.
